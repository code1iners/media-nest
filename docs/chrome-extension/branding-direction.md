# MyTube Extract 브랜딩 방향

## 목적

이 문서는 Chrome 확장 프로그램의 최소 배포 가능 브랜딩 방향을 정리한다. 구현 상세는 `current-implementation-prd.md`와 `current-implementation-fsd.md`를 기준으로 유지하고, 여기서는 제품명, 카피, 시각 톤, 정책 안내, UI 문구 원칙만 다룬다.

## 브랜드 정의

- 제품명은 `MyTube Extract`로 한다.
- 핵심 콘셉트는 사용자가 가진 YouTube watch URL을 필요에 맞게 영상 또는 음성으로 추출하는 보조 도구다.
- 브랜드 감성은 B급 밈 감성과 실제 도구로서의 명확성을 함께 가져간다.
- 원본 밈 이미지나 YouTube 공식 로고, 색 조합, 제휴처럼 보이는 표현은 직접 사용하지 않는다.

## 핵심 카피

- 메인 카피: `이 영상은 이제 제 겁니다`
- 정책 안내 문구: `저작권 및 플랫폼 정책을 준수해 사용하세요.`

메인 카피는 popup 상단에서 강하게 노출한다. 정책 안내 문구는 다운로드 전 사용자가 확인할 수 있도록 전면 영역에 둔다.

## 시각 방향

- 전체 UI는 픽셀 게임 UI 스타일로 구성한다.
- 팔레트는 16-bit 콘솔 느낌을 기준으로 한다.
- 기본 바탕은 짙은 남색 또는 회색 계열을 사용한다.
- 강조색은 노랑, 민트, 빨강 계열을 제한적으로 사용한다.
- 상단 영역은 게임의 아이템 획득 화면처럼 구성한다.
- 오리지널 도트 일러스트를 사용해 밈을 연상시키되, 원본 밈 이미지를 복제하지 않는다.

## 아이콘 방향

- Chrome 툴바 아이콘은 픽셀 집게 또는 추출기 형태를 기준으로 한다.
- 작은 크기에서도 "영상에서 무언가를 뽑아낸다"는 의미가 보이도록 단순한 실루엣을 사용한다.
- YouTube 로고, 공식 재생 버튼 형태, 공식 브랜드 색 조합을 직접 연상시키는 구성을 피한다.

## Popup 문구 원칙

- UI 문구는 우선 한국어로 작성한다.
- 다국어 처리는 후속 범위로 둔다.
- 상단, ready 상태, 성공 상태에서는 B급 밈 톤을 허용한다.
- 설정, 입력 라벨, 오류, 서버 연결 실패 문구는 명확하고 실용적으로 작성한다.

우선 적용할 기본 문구는 아래와 같다.

| 기존 문구 | 변경 문구 |
| --- | --- |
| `Download type` | `추출 형식` |
| `Audio` | `오디오` |
| `Video` | `비디오` |
| `Source URL` | `추출 URL` |
| `Filename` | `파일명` |
| `Start download` | `추출 시작` |

## 구성 방향

Popup은 아래 순서로 구성한다.

1. 제품명 `MyTube Extract`
2. 픽셀 게임 배너와 메인 카피 `이 영상은 이제 제 겁니다`
3. 정책 안내 문구 `저작권 및 플랫폼 정책을 준수해 사용하세요.`
4. 추출 URL, 추출 형식, 파일명, 세부 옵션 입력 영역
5. `추출 시작` 버튼

상단 배너는 강한 콘셉트를 담당하고, 하단 폼은 실제 작업을 정확히 수행하는 도구로 보이게 한다.

## 적용 범위

- `apps/chrome-extension` popup UI의 제품명, 카피, 라벨, 버튼 문구
- `wxt.config.ts`의 extension name과 description
- popup HTML title
- Chrome extension icon asset
- popup CSS의 색상, 테두리, 버튼, 입력창 스타일

## 제외 범위

- API 서버 이름과 계약 변경
- 다운로드 엔드포인트 변경
- 현재 탭 URL 자동 가져오기
- Chrome Web Store 등록 문구와 스크린샷 제작
- 다국어 리소스 구조 도입
- 원본 밈 이미지 사용

## 검증 기준

- popup 첫 화면에서 `MyTube Extract`와 `이 영상은 이제 제 겁니다`가 명확히 보인다.
- 정책 안내 문구가 다운로드 실행 전 사용자에게 노출된다.
- 오디오와 비디오 추출 기능이 기존과 동일하게 동작한다.
- 픽셀 게임 UI 스타일이 적용되어도 입력값과 버튼 텍스트의 가독성이 유지된다.
- 툴바 아이콘이 작은 크기에서도 추출 도구로 인식된다.
- YouTube 공식 제품 또는 제휴 서비스처럼 보이지 않는다.
